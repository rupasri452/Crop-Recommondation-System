About OptiCrop

OptiCrop is an Artificial Intelligence based Crop Recommendation System. It helps farmers select the most suitable crop by analyzing soil nutrients and climatic conditions using Machine Learning.

🌱 Soil Analysis

Analyzes Nitrogen, Phosphorous and Potassium levels.

🌦 Climate Analysis

Uses Temperature, Humidity, pH and Rainfall.

🤖 Machine Learning

Predicts the most suitable crop using Logistic Regression.

⚡ Fast Prediction

Provides crop recommendation within seconds.

Technologies Used

🐍 Python 🌐 Flask 🤖 Machine Learning 📊 Pandas 📈 NumPy 🎨 HTML & CSS
